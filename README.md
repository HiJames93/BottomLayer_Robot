# BottomLayer_Robot

效果预览
----
#### 介绍：
这是一个由树莓派+mcu组成的一个IOT网络机器人，通过TCP/IP与上层核心进行
数据交换。同时为外界提供API
----
#### 组成：
* 软件/框架：
    1. [springboot](https://spring.io/projects/spring-boot) 
    2. [mybatis](www.mybatis.org) 
    3. [sqlite3](www.sqlite.org)
    4. [pi4j](https://pi4j.com/) 
    5. c/c++
    6. Java 
    7. [shell](https://www.runoob.com/linux/linux-shell.html) 
    8. python
----
